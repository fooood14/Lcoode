# Lcood Project

مشروع واجهة مستخدم مبني باستخدام **Vite + Tailwind CSS**.

## 🚀 الإعداد المحلي
```bash
npm install
npm run dev
```

## 🛠️ البناء للنشر
```bash
npm run build
```

## 📦 النشر على Vercel
- ارفع المشروع إلى GitHub
- ثم استورده على [vercel.com](https://vercel.com)
- الإعدادات:
  - Framework: Vite
  - Build Command: vite build
  - Output Directory: dist
